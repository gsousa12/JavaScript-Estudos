# Javascript

```js
// Variaveis Globais

var DIRECTION = {
    IDLE: 0,
    UP: 1,
    DOWN: 2,
    LEFT: 3,
    RIGHT: 4
};
```

A variavel DIRECTION é um objeto que atual como uma enumeração.

DIRECTION é usada para representar as direções que podem ser utilizadas pelo player para se movimentar.
